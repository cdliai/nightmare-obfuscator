# Security Policy

Nightmare Obfuscator is intended for defensive IP sharing and artifact
provenance. Please do not use public issues for vulnerability reports.

Report security concerns to CDLI.ai maintainers at:

```text
security@cdli.ai
```

Include the affected version or commit, reproduction steps, expected impact, and
whether the issue affects generated obfuscated artifacts, manifest integrity, or
the CLI itself.

We do not treat requests for deobfuscation, malware concealment, runtime access
gating, or bypassing analysis tools as supported security use cases.
